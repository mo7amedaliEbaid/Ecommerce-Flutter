# Alpha Store
Alpha Store is a Flutter app for hyper, multi-categories market to display products, manage sales and orders.      

## State Management
- Provider

## Features
- Notifications
- Authentication
- Cache     
- Cart
- Connectivity Checking
- Loading Shimmer
- Searc functionality        
- Localization to support (English, Arabic)     
- Responsive Design
- Sorting products
